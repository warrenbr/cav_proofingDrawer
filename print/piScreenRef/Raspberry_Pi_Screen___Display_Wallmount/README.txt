# Summary

So here is my Raspberry Pi Wallmount. Perfect for your smart home projects, with plenty of space inside for other controllers ,relais, or whatever you want. You can also attach a standard 30x30x10 fan. All you need to buy are 4 countersunk M4 bolts and 4 M4 nuts, and screws to attach the display and raspberry.
(Designed for the raspberry pi 3)
If you have any ideas or recommondations for version 2, please write a comment
Have fun printing it!
